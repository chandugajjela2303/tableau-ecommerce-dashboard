🛒 Ecommerce Sales Dashboard (Tableau)
📌 Project Overview

This project presents an interactive Ecommerce Sales Dashboard built using Tableau to analyze sales performance, customer behavior, and return trends. The dashboard helps in making data-driven decisions by providing clear insights into business operations.

🎯 Objectives
Analyze overall sales performance
Identify top-performing products and categories
Track customer purchase behavior
Monitor returns and their impact on revenue
Visualize trends over time
📊 Dashboard Features
📈 Sales Overview – Total revenue, profit, and quantity sold
🛍️ Category-wise Analysis – Performance by product category
🌍 Region-wise Sales – Sales distribution across locations
🔁 Returns Analysis – Returned orders and their patterns
📅 Time Series Analysis – Monthly/Yearly trends
👥 Customer Insights – Repeat vs new customers
🧰 Tools & Technologies Used
Tableau – Data visualization and dashboard creation
Microsoft Excel / CSV – Data source
Data Cleaning – Preprocessing before visualization
📁 Dataset

The dataset used in this project contains:

Order details (Order ID, Date, Product, Category)
Customer details (Customer ID, Region)
Sales metrics (Sales, Profit, Quantity)
Returns data
